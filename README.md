# Videogame-JS

A lightweight, **Vanilla JavaScript** browser game featuring grid-based navigation and collision logic. This project demonstrates core game engine principles like coordinate-based movement, state management, and dynamic level rendering without external libraries.

## 🚀 Features
* **Grid-Based Navigation:** Precise player movement using event-driven controls.
* **Collision Logic:** Automated detection for obstacles and goal triggers.
* **Modular Level Design:** Maps are stored as data strings in `maps.js`, making it easy to add new challenges.
* **Responsive UI:** Built with **HTML5** and **CSS3** to ensure a consistent experience across different screen sizes.

## 🛠️ Technical Stack
* **JavaScript (ES6+):** Core game logic and engine.
* **HTML5:** Game container and structure.
* **CSS3:** Responsive layout and styling.

## 📂 Project Structure
* `index.html`: Main entry point.
* `games.js`: The game engine and input handling.
* `maps.js`: Level blueprints and map data.
* `styles.css`: Visual interface and grid positioning.
